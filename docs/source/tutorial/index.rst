Tutorial
========

.. toctree::
    :caption: Contents:
    :maxdepth: 1

    example

This tutorial will walk you through using Morpheus to systematically
compare two data sets.

While it's designed to give a good starting point, the tutorial doesn't
cover all of Morpheus' features. Check out the :ref:`quickstart` for an
overview of what Morpheus can do, then dive into the docs to find out more.
