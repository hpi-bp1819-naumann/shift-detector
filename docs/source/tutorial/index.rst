Tutorial
========

.. toctree::
    :caption: Contents:
    :maxdepth: 1

    example

This tutorial will walk you through using Icarus to systematically
compare two data sets.

While it's designed to give a good starting point, the tutorial doesn't
cover all of Icarus' features. Check out the :ref:`quickstart` for an
overview of what Icarus can do, then dive into the docs to find out more.
